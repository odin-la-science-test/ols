🔧 GESTION DU SERVEUR :

   Pour voir les processus actifs :
   • Utilisez l'outil Kiro "listProcesses"

   Pour arrêter le serveur :
   • Utilisez l'outil Kiro "controlPwshProcess" avec action "stop"
   • Ou fermez le terminal du serveur

   Pour redémarrer le serveur :
   • node server/emailServerResend.js
