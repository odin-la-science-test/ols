    🚀 INSTALLATION RAPIDE (5 minutes)
    ═══════════════════════════════════════════════════════════════

    ┌───────────────────────────────────────────────────────────────┐
    │  ÉTAPE 1 : Créer un Compte SendGrid (2 min)                  │
    └───────────────────────────────────────────────────────────────┘

    1. Allez sur : https://signup.sendgrid.com/
    2. Remplissez le formulaire (email, mot de passe, nom)
    3. Cliquez sur "Create Account"
    4. Vérifiez votre email et cliquez sur le lien


    ┌───────────────────────────────────────────────────────────────┐
    │  ÉTAPE 2 : Créer une Clé API (1 min)                         │
    └───────────────────────────────────────────────────────────────┘

    1. Connectez-vous à SendGrid
    2. Allez dans Settings > API Keys
    3. Cliquez sur "Create API Key"
    4. Nom : "Odin La Science"
    5. Permissions : "Full Access"
    6. Cliquez sur "Create & View"
    7. COPIEZ LA CLÉ (format: SG.xxx...)


    ┌───────────────────────────────────────────────────────────────┐
    │  ÉTAPE 3 : Installer (2 min)                                 │
    └───────────────────────────────────────────────────────────────┘

    Ouvrez PowerShell dans ce dossier et exécutez :

        .\setup-sendgrid.ps1

    Le script va :
    • Installer @sendgrid/mail
    • Vous demander votre clé API
    • Configurer automatiquement tout


    ═══════════════════════════════════════════════════════════════

    ✅ C'EST TOUT ! Maintenant :

    ═══════════════════════════════════════════════════════════════

    🚨 PROBLÈMES ?
    ═══════════════════════════════════════════════════════════════

    ❌ "SENDGRID_API_KEY non configurée"
       → Vérifiez server/.env
       → La clé doit commencer par SG.

    ❌ "Unauthorized"
       → La clé API est incorrecte
       → Créez une nouvelle clé sur SendGrid

    ❌ L'email n'arrive pas
       → Vérifiez le dossier Spam
       → Vérifiez FROM_EMAIL dans .env
