    ⚡ MÉTHODE ULTRA-RAPIDE (2 MINUTES)
    ═══════════════════════════════════════════════════════════════

    Ouvrez PowerShell et tapez :

        .\ouvrir-resend.ps1

    Le script va :
    • Ouvrir le site Resend
    • Vous guider pour créer votre compte
    • Configurer automatiquement tout


    📝 OU SUIVEZ CES 3 ÉTAPES
    ═══════════════════════════════════════════════════════════════

    1️⃣  Créer un compte (1 minute)
       https://resend.com/signup

    2️⃣  Obtenir une clé API (30 secondes)
       Dashboard → API Keys → Create API Key

    3️⃣  Installer (30 secondes)
       .\setup-resend.ps1

    ═══════════════════════════════════════════════════════════════

       npm run dev
