📧 POUR RECEVOIR DE VRAIS EMAILS (OPTIONNEL) :

Si vous voulez vraiment recevoir des emails, vous devez :

1. Créer un compte sur https://resend.com/

2. Vérifier votre propre domaine
   (ex: votresite.com)

3. Obtenir une vraie clé API

4. Modifier server/.env :
   RESEND_API_KEY=re_votre_vraie_cle
   FROM_EMAIL=noreply@votredomaine.com

5. Modifier .env.local :
   VITE_EMAIL_PROVIDER=custom

6. Redémarrer les serveurs
