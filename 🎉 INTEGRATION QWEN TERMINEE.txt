🤖 QU'EST-CE QUE QWEN2-VL ?
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

Qwen2-VL 14B est une IA vision-language qui :

✓ Analyse des images scientifiques
✓ Répond à des questions sur les images
✓ Comprend le contexte scientifique
✓ Génère des descriptions détaillées
✓ Fonctionne 100% en local (pas d'internet)
✓ Gratuit après installation
✓ Confidentialité totale

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

VISION :
  • Images de microscopie
  • Gels d'électrophorèse
  • Graphiques scientifiques
  • Spectres (IR, RMN, etc.)
  • Reconnaissance d'objets
  • Description détaillée

LANGUAGE :
  • Réponses en français
  • Contexte scientifique
  • Explications détaillées
  • Questions-réponses
  • Génération de texte


🚀 INSTALLATION (3 ÉTAPES)
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

1️⃣  INSTALLER OLLAMA
    → https://ollama.ai/
    → Télécharger et installer
    → Redémarrer l'ordinateur

2️⃣  INSTALLER QWEN2-VL
    → .\installer-qwen.ps1
    → Suivre les instructions
    → Télécharger le modèle (~8GB)

3️⃣  DÉMARRER
    → .\demarrer-tout.ps1
    → Allez sur http://localhost:3000

DEPUIS LE CODE :
┌─────────────────────────────────────────────────────────┐
│ import QwenService from '@/services/qwenService';       │
│                                                          │
│ // Analyser une image                                   │
│ const result = await QwenService.analyzeImage(          │
│   imageFile,                                             │
│   'Décris cette image'                                  │
│ );                                                       │
│                                                          │
│ // Analyser une image scientifique                      │
│ const result = await QwenService.analyzeScientificImage(│
│   imageFile,                                             │
│   'microscopy' // ou 'gel', 'graph', 'spectrum'        │
│ );                                                       │
│                                                          │
│ // Poser une question                                   │
│ const result = await QwenService.askScientificQuestion( │
│   'Qu\'est-ce que l\'ADN ?',                            │
│   'biology' // ou 'chemistry', 'physics'               │
│ );                                                       │
└─────────────────────────────────────────────────────────┘
