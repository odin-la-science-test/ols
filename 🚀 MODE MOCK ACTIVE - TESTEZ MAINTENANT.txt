   │ d) Cliquez sur "Suivant"                            │
   │                                                      │
   │ → Une notification apparaît en haut à droite        │

   └─────────────────────────────────────────────────────┘

3. CONNEXION :
   ┌─────────────────────────────────────────────────────┐
   │ a) Allez sur la page de connexion                   │
   │ b) Entrez email + mot de passe                      │
   │ c) Cliquez sur "Se connecter"                       │
   │                                                      │
   │ → Une notification apparaît en haut à droite        │
   │ → Le code à 6 chiffres est affiché                 │

Quand vous serez prêt à envoyer de vrais emails :

1. Configurez server/.env avec vos identifiants Gmail
   (voir le fichier "🚀 CONFIGURER GMAIL MAINTENANT.txt")

2. Modifiez .env.local :
   VITE_EMAIL_PROVIDER=custom

3. Démarrez le serveur email :
   .\demarrer-avec-gmail.ps1

4. Redémarrez le serveur web :
   npm run dev
