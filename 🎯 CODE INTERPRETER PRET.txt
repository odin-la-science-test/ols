🚀 COMMENT UTILISER :

1️⃣  DÉMARRER LES SERVEURS :
   
   Option A - Tout en un (RECOMMANDÉ) :
   .\demarrer-tout.ps1
   
   Option B - Séparément :
   .\demarrer-qwen.ps1    # Serveur IA (port 3002)
   npm run dev            # Serveur web (port 3000)

2️⃣  ACCÉDER AU CODE INTERPRETER :
   
   • Ouvrez http://localhost:3000
   • Connectez-vous
   • Allez dans Munin
   • Cliquez sur "Code Interpreter" (bouton avec icône </>)
   • Ou allez directement sur : http://localhost:3000/munin/code-interpreter

3️⃣  UTILISER LE CODE INTERPRETER :
   
   • Collez votre code dans l'éditeur
   • Sélectionnez le langage (Python, JavaScript, etc.)
   • Cliquez sur "Analyser le Code"
   • L'IA Qwen2-VL analysera votre code et fournira :
     - Explication du code
     - Détection de bugs potentiels
     - Suggestions d'amélioration
     - Optimisations possibles

⚠️  PRÉREQUIS POUR L'IA :

Le Code Interpreter fonctionne en deux modes :

MODE 1 - AVEC IA (Analyse complète) :
   • Nécessite Ollama installé
   • Nécessite le modèle Qwen2-VL téléchargé (~8GB)
   • Commande : .\installer-qwen.ps1
   • Puis : .\demarrer-qwen.ps1

MODE 2 - SANS IA (Éditeur uniquement) :
   • Fonctionne immédiatement
   • Éditeur de code fonctionnel
   • Pas d'analyse automatique
   • Utile pour visualiser et éditer du code

🔧 DÉPANNAGE :

❌ "Erreur de connexion au serveur IA" :
   → Le serveur Qwen n'est pas démarré
   → Solution : .\demarrer-qwen.ps1

❌ "Page non trouvée" :
   → Le serveur web n'est pas démarré
   → Solution : npm run dev

❌ "Modèle non trouvé" :
   → Qwen2-VL n'est pas installé
   → Solution : .\installer-qwen.ps1
