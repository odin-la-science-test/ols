📋 ÉTAPES SIMPLES (5 MINUTES)
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

1️⃣  CRÉER UN MOT DE PASSE D'APPLICATION GMAIL

    a) Allez sur : https://myaccount.google.com/apppasswords
    
    b) Si vous voyez "You are not authorized" :
       → Votre compte est géré par une organisation
       → Utilisez un compte Gmail personnel à la place
    
    c) Sinon :
       → Sélectionnez "Mail"
       → Sélectionnez "Autre" et tapez "Odin La Science"
       → Cliquez sur "Générer"
       → COPIEZ le mot de passe (16 caractères)


2️⃣  CONFIGURER server/.env

    a) Ouvrez le fichier : server/.env
    
    b) Remplacez ces lignes :
    
       GMAIL_USER=votre.email@gmail.com
       GMAIL_APP_PASSWORD=votre_mot_de_passe_app_16_caracteres
    
    c) Exemple :
    
       GMAIL_USER=john.doe@gmail.com
       GMAIL_APP_PASSWORD=abcd efgh ijkl mnop
    
    d) Sauvegardez


3️⃣  DÉMARRER LE SERVEUR GMAIL

    Option A - Automatique (Recommandé) :
    
      .\demarrer-avec-gmail.ps1
    
    Option B - Manuel :
    
      node server/emailServerGmail.js


4️⃣  TESTER

    a) Allez sur : http://localhost:3000
    
    b) Cliquez sur "S'inscrire"
    
    c) Remplissez le formulaire
    
    d) Vérifiez votre boîte Gmail
    
    e) Entrez le code reçu
    
    f) 🎉 Ça marche !
