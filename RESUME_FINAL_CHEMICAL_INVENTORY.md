## 🚀 Comment Utiliser

### Accès Rapide
1. `npm run dev`
2. Se connecter
3. Aller sur `/beta`
4. Cliquer sur "Chemical Inventory"

### Fonctionnalités Clés
- **Dashboard** : Clique sur les KPIs pour filtrer
- **Recherche** : Tape dans la barre de recherche
- **Filtres** : Clique sur "Filtres" pour ouvrir le panneau
- **Affichage** : Bascule entre grille et table
- **Tri** : Clique sur les en-têtes de colonnes (mode table)
- **Actions** : Ajuste le stock, duplique, supprime
- **Export** : Clique sur "Export CSV"
