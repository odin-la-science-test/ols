Problème Actuel

Erreur de chargement

Le module de messagerie ne se charge pas correctement dans l'application principale. Cela peut être dû à:

  • Le serveur WebSocket n'est pas démarré
  • Une dépendance manquante
  • Un problème d'import/export
  • Une erreur dans le lazy loading

Démarrage du Serveur WebSocket

Pour démarrer le serveur WebSocket, exécutez:

npm run messaging:server

Ou manuellement:

node src/messaging-api/server.ts

Accès au Module

Une fois le serveur démarré, accédez au module via:

http://localhost:3000/hugin/messaging
