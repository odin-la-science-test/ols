🎯 PANNEAU LATÉRAL COULISSANT (DRAWER)
   • S'ouvre depuis la GAUCHE de l'écran
   • Animation fluide et professionnelle
   • Largeur: 450px (desktop) / 85% (mobile)
   • Overlay semi-transparent pour fermer

📑 DEUX VUES DANS LE PANNEAU
   1. 📝 VUE CODE
      → Affiche le code source brut
      → Police monospace pour la lisibilité
      → Bouton copier intégré
   
   2. 👁️ VUE INTERPRÉTATION
      → Affiche l'analyse de Mimir
      → Résultats formatés
      → Bouton copier intégré

🔄 SYSTÈME D'ONGLETS
   • Basculer facilement entre Code et Interprétation
   • Onglet actif mis en évidence
   • Transition instantanée

OUVRIR LE DRAWER
────────────────
   1. Aller sur le Code Interpreter
   2. Cliquer sur le bouton "Vue Détaillée" (☰)
   3. Le panneau glisse depuis la gauche

CHANGER DE VUE
───────────────
   • Cliquer sur l'onglet "Code" → Voir le code source
   • Cliquer sur l'onglet "Interprétation" → Voir l'analyse

FERMER LE DRAWER
────────────────
   Option 1: Cliquer sur le X dans le header
   Option 2: Cliquer sur la zone sombre (overlay)
   Option 3: Re-cliquer sur "Vue Détaillée"

SCÉNARIO 1: Vérifier le code
   1. Coller du code dans l'éditeur
   2. Ouvrir le drawer → Vue Code
   3. Vérifier le formatage
   4. Copier si nécessaire

SCÉNARIO 2: Analyser le code
   1. Cliquer sur "Analyser le code"
   2. Ouvrir le drawer → Vue Interprétation
   3. Lire l'analyse détaillée
   4. Copier les suggestions

SCÉNARIO 3: Comparer code et analyse
   1. Analyser le code
   2. Ouvrir le drawer
   3. Basculer entre Code et Interprétation
   4. Comparer les deux vues
